# Babel

### babel 是什么？

- `Babel`是一个工具链，用来将最新的 js 代码转换为向后兼容的代码，从而能够运行在旧版本浏览器和其他环境中。
- `Babel` 能够转换 `jsx` 语法
  - 需要预设 `@babel/preset-react`

* 编译过程分为三个阶段
  - 解析，转换，输出

### 插件系统

- 加载时机
  - 执行时机是在`preset`之前
  - **插件的执行顺序是从前到后**
  - **`prest`的执行顺序是从后到前**
- 参数

  ```
  plugins:[
      [
          "plugin-a",
          {

          }
      ],
      [],

  ```
